# identity-theft

Steals fictional identities, not personal data. Converts your text, Markdown,
or HTML into a fictional character's voice — Ron Swanson, Yoda, Darth Vader,
GLaDOS, Mary Poppins, and 46 more — while code blocks, commands, links, and
facts survive untouched. No actual crimes committed.

Install with [skillshare](https://github.com/runkids/skillshare), or copy
`skills/identity-theft/` into your skills directory.

**OpenAI Codex / ChatGPT:** the skill format is compatible as-is — copy this
directory into `~/.agents/skills/` (or your project's `.agents/skills/`) and
Codex discovers it via the same `SKILL.md`. See OpenAI's
[Build skills](https://learn.chatgpt.com/docs/build-skills) guide.
