# Support

## Questions and help

- **Using a skill?** Start with the skill's own `README.md` and `SKILL.md`
  under `skills/<name>/`.
- **Something broken, or a feature idea?** Open an
  [issue](https://github.com/rubicon/ai-skills/issues). Search first — it may
  already be filed.
- **Want to add a personality to `identity-theft`?** Use the "Propose a
  personality" issue template, then see [CONTRIBUTING.md](CONTRIBUTING.md).

## What this project is

A curated collection of AI skills (reusable prompt templates for Claude Code
and compatible agents, including OpenAI Codex) and Claude Code plugins. It is
maintained on a best-effort basis. There is no SLA, but issues and PRs are
read.

## Security

For anything sensitive, do not open a public issue — see
[SECURITY.md](SECURITY.md).
