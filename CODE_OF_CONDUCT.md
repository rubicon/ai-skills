# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1. The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: be respectful. Harassment, discrimination, and personal attacks are
not welcome. Contributors are expected to act professionally and in good faith.

## Enforcement

Report unacceptable behavior to the maintainer at **dax@daxdavis.com**. Reports
will be handled discreetly. Maintainers may remove, edit, or reject
contributions, comments, and contributors that violate this Code of Conduct.
